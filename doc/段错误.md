segment fault:不正确的指针访问,指针一定要指向空间,才能操作
- 空指针访问
```
int *p = NULL;
*p = 1;
```
- 野指针访问
- 数组越界

## 内存转储(日志文件)
- 在运行的shell里面运行命令
```
ulimit -c unlimited
```
(在shell关闭重启后,要重新运行一次)
- 运行程序<br>
程序发生错误是,程序退出,同时当发生错误时的信息转储(dump)到core.*文件
- 用gdb来查看发生段错误的代码位置
```
gdb hello core.***
```
